# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language.
- Being respectful of differing viewpoints and experiences.
- Gracefully accepting constructive criticism.
- Focusing on what is best for the community.

Examples of unacceptable behavior include:

- Harassment, intimidation, or discriminatory language.
- Personal or political attacks.
- Publishing others' private information without explicit permission.
- Other conduct that could reasonably be considered inappropriate in a professional setting.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this code of conduct.
