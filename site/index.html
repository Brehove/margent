Local-first Markdown review for Mac

Margent

Write in Markdown, anchor threaded comments to passages, and review Codex or Claude suggestions without sending your workspace to a web app.

Download for Mac View on GitHub

Downloads are served from the latest GitHub Release.

Review Markdown where it already lives.

Margent keeps documents as ordinary files and stores review state in local .mdreview sidecars. Your comments, proposals, and provider replies stay beside the draft.

Built for agent-assisted editing.

Use Codex or Claude Code through your own installed CLIs. Margent shows replies and revision proposals in the app while preserving the source Markdown.
